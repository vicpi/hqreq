# HQReq

## Installation

Using npm:

```bash
$ npm install --save hqreq
```

## Usage

#### get
```javascript
HttpRequest.get('/books')
  .then(function (response) {
    // Get list of books in response
  })
  .catch(function () {
    // Handle error response
```
